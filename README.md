# 🏦 Bank System

## 📌 Description
Desktop banking system built using **C# WinForms** and **SQL Server**.  
The system is designed to manage customers, accounts, and basic banking operations in a simple desktop environment.

This project is intended for learning, practice, and portfolio purposes.

---

## ⚙️ Technologies Used
- C#
- WinForms
- SQL Server
- ADO.NET
- .NET Framework

---

## ✨ Features
- Customer management
- Account creation
- Deposit and withdrawal operations
- Transaction tracking
- Basic reports
